# URBNSTEP 品牌簡介網站

一個展示 URBNSTEP 街頭機能鞋品牌故事的響應式網站。

## 專案特色
- 現代化的品牌故事展示
- 清晰的品牌理念與歷程介紹
- 響應式設計，適配各種裝置
- 優化的字體載入與效能

## 技術架構
- **HTML5** - 語義化標記
- **CSS3** - 現代化樣式與佈局
- **Google Fonts** - Noto Sans TC 字體
- **Bootstrap Icons** - 可縮放向量圖示
